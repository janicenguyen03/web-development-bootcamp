Selecting HTML Elements with Javascript
    getElementsByTagName("li") -> array, behave as an array
    getElemetsByClassName("btn") -> array, even only one item
    getElementByID("title") -> return one item only
    querySelector("") -> return one item only | selector: selector as in css
        querySelector(".btn")
        querySelector("#title")
        querySelector("li a")

        get back the first item in the document that satisfy the selector
    querySelectorAll("#list .item") | array

    document.querySelector("li a").style.color="red";
